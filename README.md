# 🦫 RUN, CAPY, RUN!

An endless runner game featuring a brave capybara escaping through a mystical jungle.

## Features

- Endless runner gameplay
- Double jump mechanics
- Dynamic difficulty scaling
- Local leaderboard system
- Beautiful animated jungle environment
- Mobile and desktop support
- Retro arcade aesthetic

## Controls

| Action | Key |
|----------|------|
| Jump | SPACE |
| Jump | TAP |
| Leaderboard | L |

## Play Online

Enable GitHub Pages and play directly in your browser.

## Installation


A retro-style endless runner where a fearless capybara escapes through a magical jungle filled with obstacles, coins, and leaderboard challenges.
